# Mindful Scroll Check

A Chrome extension that helps you stay mindful while browsing short-form content platforms like YouTube Shorts and TikTok.

## Features

- **Mindfulness Prompts**: Every 5 minutes, you'll be prompted with "Are you enjoying your time right now?"
- **Automatic Tab Closure**: If you select "No", the tab automatically closes to help you get back to work
- **Response Tracking**: Your Y/N responses are saved locally for personal reflection
- **Works on**: YouTube Shorts and TikTok

## Installation

1. Download or clone this repository
2. Open Chrome and navigate to [chrome://extensions/](chrome://extensions/)
3. Enable "Developer mode" (toggle in the top right)
4. Click "Load unpacked" and select this extension's directory
5. The extension will automatically activate on YouTube Shorts and TikTok pages

## How It Works

When you visit YouTube Shorts or TikTok, the extension starts a 5-minute timer. After 5 minutes of scrolling, a modal appears asking if you're enjoying your time. Your response helps you stay conscious of your browsing habits and make intentional decisions about your screen time.

## Privacy

All data is stored locally in your browser. Nothing is sent to external servers.
